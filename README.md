This Quiz Builder and Quiz Player is an app created with React, Vite, TypeScript and Tanstack Query v5. 
It showcases how coding quizzes can be made and how it would be fetched when a quiz id is created. 


## Installation 

```js
npm install

```

##  Running the app locally

```js

npm run dev

```

## Accessing the app locally
```js

localhost:5173

```

## Running the Storybook

```js

npm run storybook

```

## Accessing the Storybook
```js

localhost:6006

```

# Design Process
- Branding, I focused on creating a quiz builder and player for Bookipi and kept the brand's logo and accent colors for the design. 
- The logo for the app is simple with Bookipi logo and a plain text for Quiz.
- Accent Colors applied thru all the design components that showcase a unified UI/UX design specifically for Bookipi.

# Technical Decisions
- Tech stack that were used is React.js + Vite + Storybook + TypeScript + TanStack Query + Radix UI + TailwindCSS
- Third-party libraries were used for other components
- UI/UX were designed with Figma, no prior UX research were done but kept in mind the branding for Bookipi
- API that was given also served as the data source for the Quizzes' data. 

# Areas of Improvement
- As this exercise was only done on a 7-day period there are areas of improvement to make this app scalable and can be used for production
- Additional Unit and UI/UX Testing should be added to ensure production-readiness
- Additional monitoring tools should be added
- Refining of the UI/UX Design
- Add additional stories for all the components
