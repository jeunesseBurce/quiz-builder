# Quiz Builder x Bookipi

This Quiz Builder is an app created with React, Vite, TypeScript and Tanstack Query v5. 
It showcases how coding quizzes can be made and how it would be fetched when a quiz id is created. 


## Installation 

```js
npm install

```

##  Running the app locally

```js

npm run dev

```
